# Questions

## Em thiết kế logging như thế nào?

Em dùng structured logging theo JSON format, attach traceID vào từng request, và phân loại log thành HTTP log, SQL log và error log.
Hiện tại lưu file để demo, nhưng production em sẽ push về một centralized system như Loki hoặc ELK để query và cảnh báo dễ hơn.

## Em theo dõi slow query hay memory leak như nào?

PostgreSQL: pg_stat_activity, pg_stat_statements.
App: export metrics qua Prometheus → Grafana để xem latency, goroutine count, memory profile.
Nếu production em dùng thêm OpenTelemetry để trace từng request.
